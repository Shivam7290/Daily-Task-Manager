# 📝 Daily Task Manager

A simple and responsive Daily Task Manager built using HTML, CSS, and JavaScript. This application helps users manage their daily tasks by adding, completing, and deleting tasks.

## 🚀 Live Demo

https://daily-task-manager-kappa.vercel.app/

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and user-friendly interface
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

```bash
Daily-Task-Manager/
│
├── index.html
├── style.css
└── script.js
```

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Shivam7290/Daily-Task-Manager.git
```

2. Open the project folder

```bash
cd Daily-Task-Manager
```

3. Open `index.html` in your browser
